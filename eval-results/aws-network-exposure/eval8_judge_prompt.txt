### Option 1: AWS SSO (most common for orgs)
```bash
aws configure sso
# Follow prompts: SSO start URL, region, account, role
aws sso login --profile <your-profile>
```

### Option 2: Access Key + Secret
```bash
aws configure
# Prompts for:
#   AWS Access Key ID
#   AWS Secret Access Key
#   Default region (e.g. ap-south-1)
#   Output format (json)
```

### Option 3: Existing named profile
```bash
# List what's already configured
cat ~/.aws/credentials
cat ~/.aws/config

# Use a specific profile
export AWS_PROFILE=my-profile
aws sts get-caller-identity  # verify it works
```

### Option 4: Temporary session token
```bash
aws configure set aws_access_key_id ASIA...
aws configure set aws_secret_access_key ...
aws configure set aws_session_token ...
```
